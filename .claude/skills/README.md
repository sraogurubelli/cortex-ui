# Cortex UI Skills

Small, focused skills for building WebChatUI apps with Cortex UI (Left Nav, Chat, List, Dashboards).

## Skills

| Skill | When to use |
|-------|-------------|
| **webchatui-framework** | Planning or scoping; where to put Left Nav, Chat, List, Dashboards; what to add to @cortex/core vs apps. |
| **design-system-basics** | Using design system components and tokens (Layout, Text, Button, cn- tokens); generic verification. |
| **left-nav-sidebar** | Building app shell, sidebar, nav items, collapse/rail, main content area. |
| **chat-ui** | Building chat UIs with @cortex/core (Chat, ChatInput, ChatPanel, MessageBubble, etc.). |
| **list-and-dashboards** | List views (card grids, DataTable), dashboards (stat cards, metrics, grids). |
| **ui-guidelines** | Generic component verification, do/don’t examples, type safety; product-agnostic. |

## WebChatUI pillars

1. **Left Nav** – Sidebar + main area (see left-nav-sidebar).
2. **Chat** – @cortex/core chat components (see chat-ui).
3. **List** – Card grids, tables, empty states (see list-and-dashboards).
4. **Dashboards** – Stat cards, metrics, overview layout (see list-and-dashboards).
